# 屏幕覆盖模块

这是一个LSPosed模块，用于在屏幕上显示一个黑圈来覆盖摄像头位置的漏液区域。

## 功能

- 在屏幕上显示一个黑色圆形覆盖层
- 可调整覆盖层的位置和大小
- 支持保存配置
- 需要悬浮窗权限

## 安装和使用

### 1. 编译APK

```bash
./gradlew assembleDebug
```

生成的APK位于 `app/build/outputs/apk/debug/` 目录。

### 2. 安装模块

1. 将APK安装到手机上
2. 在LSPosed管理器中启用此模块
3. 选择需要作用域的系统应用（如SystemUI）

### 3. 配置覆盖层

1. 打开"屏幕覆盖"应用
2. 调整滑块设置黑圈的位置和大小
3. 点击"保存设置"
4. 点击"启动覆盖"开始显示

### 4. 权限

首次使用时需要授予悬浮窗权限：
1. 点击"启动覆盖"
2. 系统会提示授予权限
3. 在设置中开启"允许显示在其他应用上层"

## 技术说明

- 使用WindowManager添加系统级覆盖窗口
- 通过SharedPreferences保存配置
- 默认位置在屏幕中央
- 覆盖层不可触摸，不会影响正常操作

## 注意事项

- 覆盖层会显示在所有应用之上
- 重启手机后需要重新启动覆盖层
- 如果覆盖层位置不准确，请重新调整并保存
